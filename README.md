# EH_INFO

## INTRO

一个 Python 爬虫程序，用来获得 E 站漫画信息。

## USAGE

```bash
git clone https://github.com/Avimitin/EH_info.git
# 需要 Python 3.6+ 环境
# 安装依赖
pip3 install -r requirements.txt
# 执行程序
python3 main.py -u 'https://example.com'
```

程序将返回一个含有漫画基础信息的字典。
